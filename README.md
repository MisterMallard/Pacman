# PAC in Time 

## Je bent een tijdreiziger die met zijn tijdmachine is neergestort in de prehistorie! Verzamel alle ontbrekende onderdelen van de machine om terug naar huis te gaan, maar pas op, want er zijn dinosaurussen op jacht...

# De Doel

## Het doel van het spel is om alle gebroken stukken te verzamelen om het spel te voltooien zonder door de dinosaurussen gepakt te worden.

# Specials 

## Verspreid over de kaart vind je gadgets die je via de tijdmachine hebt gekregen. Je kunt ze allemaal inzetten en elk heeft een andere functie.
